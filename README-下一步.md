# AI生意工具站：上线和变现说明

## 你现在拿到的是什么

这是一个可以直接打开的动态网站原型，不需要安装后端：

- `index.html`：网站页面结构
- `styles.css`：页面样式和手机端适配
- `app.js`：工具筛选、收藏、收益估算、弹窗、表单示例数据
- `assets/`：首页看板图和赞助位图片

## 怎么打开

双击 `index.html`，或把整个文件夹上传到 Vercel、Netlify、Cloudflare Pages 这类静态网站平台。

## 怎么开始赚钱

1. 先选细分方向  
   建议先做“AI写作工具”“短视频AI工具”“跨境电商AI工具”“本地商家AI工具”中的一个，不要一开始什么都收录。

2. 每周更新内容  
   把 `app.js` 里的 `tools` 数组替换成真实工具，每个工具配一篇教程或测评文章。广告联盟通常更看重原创、有用、能吸引用户的内容。

3. 接广告或赞助  
   把页面右侧“赞助位”卖给 AI 工具、课程、SaaS 服务、本地商家。前期可以先按周收费，流量稳定后按月收费。

4. 接付费会员  
   会员内容可以先做轻量版：提示词模板、行业工具清单、短视频脚本模板、SEO选题表、自动化流程包。

5. 接表单服务  
   当前表单是本地保存演示。正式上线可以接 Formspree、Tally、飞书表单、金数据、腾讯问卷或你自己的后端接口。

6. 接支付链接  
   没有后端也可以先用 Stripe Payment Links、微信收款码、支付宝收款码、知识星球、小鹅通等方式收款。把按钮链接替换成你的真实付款链接即可。

## 推荐上线顺序

1. 买域名。
2. 上传这个文件夹到 Vercel 或 Netlify。
3. 先发布 20 个真实工具和 5 篇原创教程。
4. 加统计工具，观察哪些页面有人看。
5. 有稳定访问后再申请广告联盟。
6. 同步开放赞助位和会员意向收集。

## 需要替换的位置

- 工具数据：`app.js` 里的 `tools`
- 网站名称：`index.html` 里的“AI生意工具站”
- 赞助图片：`assets/sponsor-banner.png`
- 支付链接：`index.html` 里会员按钮的链接
- 表单提交：`app.js` 里 `data-lead-form` 的提交逻辑
